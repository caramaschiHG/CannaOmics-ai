# Roadmap

CannaOmics AI is being developed in phases. Our goal is to build a robust, open-source framework for Cannabis genotype-to-chemotype analysis.

| Phase | Description | Status |
|-------|-------------|--------|
| **Phase 0** | **Repo Setup** <br> Foundation, documentation, CI/CD, project scaffold. | ✅ Complete |
| **Phase 1** | **Dataset & Baseline MVP** <br> Synthetic mini-dataset, compound normalization, baseline ML models, and automated report generation (`cannaomics demo`). | 🔄 In Progress |
| **Phase 2** | **Public Data Integration** <br> Integrates real public data (e.g., Dryad Watts 2021), builds the feature matrices, and runs genuine models. | ⬜ Planned |
| **Phase 3** | **Terpene Synthase (TPS) Focus** <br> Curated table of CsTPS genes, mapping variants to gene windows, model focused on terpene targets. | ⬜ Planned |
| **Phase 4** | **Cannabinoid Pathway Focus** <br> Mapping THCAS/CBDAS/CBCAS, chemotype classification models (THC/CBD dominant). | ⬜ Planned |
| **Phase 5** | **Deep Interpretability** <br> Permutation importance, SHAP, and candidate ranking engine. | ⬜ Planned |
| **Phase 6** | **Public Demo & Polish** <br> Beautiful visual reports, polished CLI experience, optimized dependencies. | ⬜ Planned |
| **Phase 7** | **Preprint / Technical Report** <br> Publish methodology and initial baseline findings. | ⬜ Planned |
